Multiple Coin Toss JS Model

This simulation uses Easy Java Simulations (Ejs) to model the problem of the tossing of 10 coins.  The result of each set of coin tosses is shown by the image of the pennies on the screen and the complete results of the tossing experiment is shown on a graph of the average number of heads vs. the total number of times the 10 coins were tossed.

This simulation was created by  Wolfgang Christian and Mario Belloni of Davidson College using the Easy Java Simulations (Ejs) modeling tool.
