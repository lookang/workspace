The average displacement for 8 random walks on a two-dimensional lattice. Data points are taken every 100000 steps.

Synchronized Parallel Computation of a 2D Random Walk

A Parallel Region is the simplest parallel programming Element and we use it to perform a two-dimensional random walk on a lattice. Although there are more efficient approaches, this simple example computes each random walk in a separate thread to demonstrate how computational performance depends on the number of threads and how the computational load is distributed among processor cores. In this simulation, a 2016 iMac with a quad-core Intel processor performs over 20000 random steps per millisecond.

Each thread performs a given number random steps and waits for other threads to complete their iterations before computing the average displacement. The average displacement of eight parallel walks is shown above. Theory predicts that the average displacement is proportional to the square root of the number of random steps.

The Synchronized Parallel Region implementation of the Random Walk on a lattice uses the EJS adaptation of the Parallel Java library developed by Alan Kaminsky. Addition examples demonstrating how to parallelize the random walk simulation using a parallel loop are available in the Open Source Physics Collection in ComPADRE.

User Interface

Main Window Bottom

  • Buttons
    • Play/Pause: Starts and stops the simulation.
    • Reset: Clears data and restores the default parameter values.
  • Threads:Number of processor threads. The initial value is twice the number of processor cores minus two.
  • Steps/point:Number of random steps between screen updates.
  • Show < r(t) >: Shows the average displacement.

Main Window Top

The initial wave function and potential are set using the drop down menu at the bottom. Select the Custom option to edit these fields.

  • # steps: Total number of random steps performed by all threads.
  • steps/ms: Total steps/milliseconds performed by all threads.
  • steps/ms/thread: Average number of random steps/millisecond performed by each thread.
