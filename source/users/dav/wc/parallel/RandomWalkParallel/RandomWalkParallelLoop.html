The trajectories of 500 random walks on a two-dimensional lattice. Data points are taken every 10000 steps.

Parallel Loop Computation of a 2D Random Walk

A Parallel Loop produces the fastest two-dimensional random walk with a large number of particles. In this simulation, a 2016 iMac with a quad-core Intel processor computes over 50000 random steps per millisecond.

The Parallel Loop iterates to advances the position N random walks. Each thread is assigned a different subrange (chunk) and performs a given number of random steps before iterating to next walk in the chunk. If a thread completes its chunk, it is assigned additional chunks until all iterations have been performed. This assignment is done automatically by the Parallel Loop scheduling algorithm and guarantees that all processor cores are fully utilized at all times.

The Parallel Loop implementation of the Random Walk on a lattice uses the EJS adaptation of the Parallel Java library developed by Alan Kaminsky. Addition examples are available in the Open Source Physics Collection in ComPADRE.

User Interface

Main Window Bottom

  • Buttons
    • Play/Pause: Starts and stops the simulation.
    • Reset: Clears data and restores the default parameter values.
  • Threads:Number of processor threads. The initial value is twice the number of processor cores minus two.
  • Steps/point:Number of random steps between screen updates.
  • Show
    • trajectories: Shows trajectories in the position plot. In order to improve performance, only the last 1000 positions are shown in each trajectory and only the first 1000 trajectories are shown.
    • < r(t) >: Shows the average displacement graph.

Main Window Top

The initial wave function and potential are set using the drop down menu at the bottom. Select the Custom option to edit these fields.

  • # steps: Total number of random steps performed by all threads.
  • steps/ms: Total steps/milliseconds performed by all threads.
  • steps/ms/thread: Average number of random steps/millisecond performed by each thread.
  • <r>: The average displacement from the random walk start at (0, 0).
